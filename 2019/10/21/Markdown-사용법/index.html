Headers

문서 제목(Title)

이것은 문서 제목입니다 h1 태그와 동일합니다


문서 부제목(Sub title)

이것은 문서 부제목입니다 h2 태그와 동일합니다

이것은 h1 입니다

Block Quote

블록 인용 문자 > 를 사용한다

이것은 블록인용 문자 입니다


중첩해서 사용이 가능하다

이것은 첫 번째 블록인용 문자 입니다

이것은 두 번째 블록인용 문자 입니다

이것은 세 번째 블록인용 문자 입니다


인용문 안에 다른 Markdown 요소를 넣을 수 있다

이것은 인용문 안의 h5 태그 입니다

this is code block


목록

순서 있는 목록(번호)

  1. first
  2. second
  3. third

어떤 번호를 입력해도 내림차순으로 정렬된다

  1. first
  2. third
  3. second

순서 없는 목록(글머리 기호)

  • first
    • second
      • third
  • first
    • second
      • third
  • first
    • second
      • third
  • 깊이를 조절하기 위해 tap을 사용한다

혼합해서 사용하는 것도 가능하다

  1. first
    1. second
      1. third

Code

Code Blocks

코드 첫줄과 마지막 줄에 ``` 혹은 ~을 붙여 사용한다

이런 식으로 언어를 지정하는 것도 가능하다
